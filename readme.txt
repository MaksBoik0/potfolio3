ПОРТФОЛИО. БАЗА ДАННЫХ ПРОЕКТ. РУКОВОДСТВО

Для того, чтобы активировать проект, необходимо:
	1) Создать четыре папки: __pycache__ , instance, static, templates.
	2) В папку __pycache__ поместить файл main.cpython-311.pyc
	3) В папку instance поместить файл diary.db
	4) В папкe static создать ещё две папки: css и img.
	5) В папку css поместить файл style.css
	6) В папку img поместить .svg и .png файлы
	7) В папку templates поместить файлы .html
	8) Без папок должны остаться main.py, Pipfile, Pipfile.lock файлы. Их поместить в общую папку, к папкам.
